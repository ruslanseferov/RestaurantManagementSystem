## 🎯 Overview

This application provides a complete restaurant management solution that allows you to:
- Manage multiple restaurant branches
- Track tables and their capacity
- Maintain menu items with pricing
- Process and track customer orders
- View comprehensive statistics and analytics

## ✨ Features

### Restaurant Management
- Add, update, and delete restaurant branches
- Track branch codes and identify each restaurant
- Monitor total orders and revenue per restaurant

### Table Management
- Create and manage dining tables per restaurant
- Set table capacity
- Track order count per table

### Menu Management
- Add menu items to restaurants
- Organize items by category
- Manage pricing
- Track total items sold

### Order Management
- Create orders with multiple menu items
- Associate orders with tables and restaurants
- Track order dates and total amounts
- Manage order items with quantity and pricing snapshots

### Statistics & Analytics
- View restaurant status and performance metrics
- Restaurant rankings by revenue and orders
- Top-selling menu items per restaurant
- Global top-selling items across all restaurants

## 🛠️ Technology Stack

- **Language**: C# (.NET)
- **Database**: SQL Server (SQLEXPRESS)
- **ORM**: Entity Framework Core
- **Architecture Pattern**: Repository Pattern with Services Layer
- **UI**: Console-based CLI
